## 🚀 Setting up the application

The first time you open iContainer, you choose your primary working folder. This folder acts as the storage area for your container data. 

1. Select "Settings" from the top navigation bar.
2. Click "Define Storage Path" to choose a directory.
3. Confirm your choice to allow the app to scan your system.
4. Restart the app if prompted to finish the configuration.

## ⚙️ Managing your containers

The main dashboard shows all active containers on your system. Every entry displays the name, current status, and disk usage rate. 

*   **Starting a container:** Click the green Play icon next to the container name. This puts the workload in a running state.
*   **Stopping a container:** Select the square stop button to halt the individual process.
*   **Detailed view:** Double-click any container to view logs and event history. This provides information on why a process might stop unexpectedly.
*   **Refresh:** Tap the circle icon at the top to update the status of all containers.

## 📋 Frequently asked questions

**Can I run multiple containers at the same time?**
Yes. You can manage several containers simultaneously. Keep in mind that each container uses system memory and processing power.

**Is my data safe during a crash?**
The software saves container states into a cache file automatically every minute. This prevents data loss during unexpected system interruptions.

**Do I need a special account to use this?**
No. This application works offline. You do not need to sign in or connect to external servers to manage your local workloads.

**How do I delete a container?**
Right-click the container entry and select "Remove." This action deletes the container and its associated temporary files. Please note that this action is permanent.
